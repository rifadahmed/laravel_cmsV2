Instructions to run a Laravel project on machine

  • go to C:\Windows\System32\drivers\etc open hosts as an adminstration paste the below linnes: 127.0.0.1 localhost 127.0.0.1 laravel.test
  • go to C:\xampp\apache\conf\extra open httpd-vhosts.conf as an adminstration paste the below lines:
    <VirtualHost *:80> DocumentRoot "C:/xampp/htdocs/laravel_test/public" ServerName laravel.test </VirtualHost>
  • Open xammp start apache and MySQl server > Create DB on PHPmyAdmin>import the Sql file located in this project folder to the DB. > Go to the given URL (EX:laravel.test) .Check the .env file for db,user name and passsword field.

    Instruction for Hosting a laravel project in Cpanel

  • Go to file manager and upload the whole project.Move all the files from project's public folder to Public_html folder. For subdomain hosting move the files directly to the subdomain hosting folder. subdomain folder will be found in the file manger folder. It will look like cms.rifadahmed.com. Now open the index.php file.Edit these two lines require __DIR__.'/../bootstrap/autoload.php'; $app = require_once __DIR__.'/../bootstrap/app.php'; To require __DIR__.'/../your_project folder_name/bootstrap/autoload.php'; $app = require_once __DIR__.'/../your_project folder_name/bootstrap/app.php';
  • Now open mysqldatabases from Cpanel and create a Database.Note down the DB,User name and password. Now open .env file from project folder and update these fields APP_URL=your website URL DB_DATABASE=name of the DB you created in Cpanel DB_USERNAME=name of the user associated with that database DB_PASSWORD=user's password Now open PhpMyAdmin from Cpanel and import the sql file from pc.
